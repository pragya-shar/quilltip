# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Summary

- Be respectful and constructive
- No harassment, trolling, or personal attacks
- Focus on what is best for the community
- Accept constructive criticism gracefully

## Enforcement

Unacceptable behavior can be reported to the project maintainers at
**<conduct@quilltip.me>**.

Reports will be reviewed and investigated, and will result in a response deemed necessary and appropriate.
